CombinedTechsCreation
- Contains data and scripts to create a table with locations and capacity factors of 
  combined hydropower and FPV plants
- Data on solar potential from https://re.jrc.ec.europa.eu/pvg_tools/en/#HR
- Data on hydropower reservoirs from Sterl, 2022 (https://doi.org/10.12688/openreseurope.13392.3)
- output: CombinedHydroSolar.xlsx

InputFilesCreation
- Contains data and scripts to create the input files for OSeMOSYS 
- main: contains all data and scripts to build and analyse the main parameters input file (.xlsx)
- other input files: contains data for the other input files (.csv)

EvapAnalysis
- Contains data and scripts to perform the evaporation analyses
